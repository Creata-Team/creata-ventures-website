01

Рынок Центральной Азии

Ни одна глобальная или региональная платформа не фокусируется на казахоязычном контенте. YouTube — хаотичный дефолт, Yandex Music — русскоязычный, Spotify — глобальный без КЗ-фокуса. Белое пятно = возможность.

Приложение Доля рынка КЗ Язык контента Социальные функции Казахский фокус
YouTube ~70-80% Все Комментарии Нет
Yandex Music 22% Русский Слабые Частично
Zvuk 23% Русский Минимальные Нет
Spotify ~10% Глобальный Слабые Нет
MuzzzApp Target Казахский + РУ + УЗ Полный соц. слой ДА — first mover
Ключевой инсайт: JioSaavn (Индия), Anghami (MENA), Boomplay (Африка) — все победили глобальных игроков через глубину локального каталога + телеком-партнёрства + национальную идентичность. MuzzzApp следует тому же playbook.

Архитектурный инсайт: Ни один open source проект не покрывает весь скоуп MuzzzApp. Рекомендуется композитная архитектура: audio engine (react-native-track-player) + plugin system (MusicFree) + creator layer (Alonetone) + social feed (Openwhyd/Funkwhale) + video (PeerTube P2P) + metadata (MusicBrainz + LRCLIB).

Vibe Coding — Инструменты

React Native + Expo
Рекомендация #1
AI-поддержка: отлично Expo EAS: no Xcode react-native-track-player Шаблоны: много
Cold start 2.8s, 48-60fps. Лучшая совместимость с AI-инструментами (Cursor, Bolt, Claude Code). Огромная экосистема шаблонов. react-native-track-player — battle-tested.
Flutter
Альтернатива #2
Performance: лучше just_audio + audio_service AI: хорошо, не отлично
Cold start 2.1s, 60fps. Pixel-perfect UI. Лучше для визуализаций и анимаций. Но Bolt/v0/Lovable не поддерживают Flutter.
